## Getting Started

1. **Clone this repo**  
    ```bash
    git clone https://github.com/danielqvu/randomOpenGLstuff.git
    cd randomOpenGLstuff/fastLearningOpengl
    ```

2. **Build with CMake**  
    ```bash
    mkdir build && cd build
    cmake ..
    make
    ```

3. **Run the app**  
    ```bash
    ./app
    ```
